Falcon Update — Januar 2024

Enhanced interface, integration with OpenAI, evaluations using RAG, API speed improvements, and streamlined Docker setup.

Ethan Sawyer

Greetings everyone 👋, take a moment to explore the latest features introduced in Januar:

  • Enhanced interface

  • Integration with OpenAI SDK (Python)

  • Guidebook for RAG evaluations in collaboration with Ragas

  • API performance enhancement

  • Streamlined Docker setup

  • SDK enhancements

    • SDKs now reveal trace URL
    • Python SDK compatibility with Pydantic versions 1 and 2

  • User experience enhancements

    • Enhanced navigation through traces
    • Single Sign-On via Google and GitHub
    • Project deletion feature

  • Documentation enhancements: Instantly access all Python documentation as Jupyter notebooks

... accompanied by various minor enhancements and bug resolutions.
